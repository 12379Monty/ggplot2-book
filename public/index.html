ggplot2 book

The ggplot book will be published by Springer, hopefully in early 2009. Until then, the latest draft version of the book will be available here. If you do download and read it, please send me your comments and criticism, so that I can improve it before it goes to press. This version of the book was produced on 16 October 2008.

ggplot2 is changing a little as I write the book, so don't be surprised if the code in the book doesn't work for you. The version on CRAN will be updated soon.
